# MvLink Notifications — SMS Consent

## Consent to Receive Messages

By using MvLink, you consent to receiving automated SMS notifications from the MvLink application to the phone number you provide.

- **Application**: MvLink — a personal health and activity monitoring app
- **Purpose**: Health reminders, activity alerts, and wellness notifications
- **Frequency**: Up to 100 messages per month
- **Opt-out**: Reply STOP to any message to unsubscribe at any time
- **Help**: Reply HELP for assistance

## Message Types

- Health and activity reminders
- Sleep and wake time alerts
- Daily wellness check-ins

## Sample Messages

- "Alert: You haven't left home today. Time to get moving!"
- "You haven't eaten today. You need to!"
- "It's late — you should be sleeping!"

Message and data rates may apply. Message frequency varies.
